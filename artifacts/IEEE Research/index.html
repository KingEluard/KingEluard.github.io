IEEE Research: Disaster Damage Assessment

Ground-level Post-Disaster Image Classification using DenseNet201 for Disaster Damage Assessment - Published in IEEE and Presented at CyMaEn 2023

Conference: January 26-27, 2023
Virtual Presentation from Philippines
IEEE Publication & Best R&D Award

Research Overview

This groundbreaking research develops an AI-powered mobile application for rapid disaster damage assessment using deep learning. Published in IEEE and presented at the International Conference on Cyber Management and Engineering (CyMaEn 2023), this work addresses the critical need for quick and accurate building damage evaluation following natural disasters to enable faster emergency response and resource allocation.

Research Objectives

  • Develop an intelligent image classification system using DenseNet201 architecture
  • Create a practical mobile application for field deployment by emergency responders
  • Achieve high accuracy in distinguishing between partially and totally damaged buildings
  • Enable rapid post-disaster damage assessment for faster response times
  • Support emergency management agencies in resource allocation decisions
  • Contribute to disaster preparedness and response strategies

Research Achievements

90% Classification Accuracy
707 Neural Network Layers
2 Damage Categories

Methodology & Technical Framework

Data Preprocessing

  • Image data augmentation techniques
  • Dataset preprocessing and normalization
  • Ground-level post-disaster image collection
  • Quality assurance and validation

DenseNet201 Architecture

  • Pretrained convolutional neural network
  • 707-layer deep learning architecture
  • Transfer learning implementation
  • Feature extraction and fine-tuning

Model Training

  • Binary classification approach
  • Global average pooling implementation
  • Dropout layer for regularization (20%)
  • Logit function for final classification

Mobile Deployment

  • Real-time image classification
  • User-friendly mobile interface
  • Field deployment optimization
  • Emergency response integration

Key Research Findings & Impact

High Classification Accuracy

Achieved 90% accuracy in distinguishing between partially damaged and totally damaged buildings, demonstrating the effectiveness of DenseNet201 architecture for disaster damage assessment.

Real-World Application

Successfully implemented the model in a mobile application capable of real-time image classification, enabling field deployment by emergency response teams.

Rapid Assessment Capability

Significantly reduced the time required for damage assessment from hours to minutes, enabling faster resource allocation and emergency response decisions.

Transfer Learning Effectiveness

Demonstrated the power of transfer learning and fine-tuning techniques in adapting pre-trained models for specialized disaster assessment tasks.

Emergency Management Support

Provided valuable insights for emergency management agencies to improve disaster response strategies and resource allocation protocols.

Scalable Solution

Created a scalable AI solution that can be adapted for various disaster scenarios and geographical regions worldwide.
